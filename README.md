# 🧮 C++ Mental Math Quiz

A modular, console-based mental math game designed to test and improve arithmetic speed. This project focuses on **clean code principles**, **struct-based data management**, and **procedural programming** in C++.

---

## 🚀 Features
* **Multiple Difficulty Levels:** Easy, Medium, Hard, and a "Mixed" mode for a dynamic challenge.
* **Diverse Operations:** Addition, Subtraction, Multiplication, Division, and a Random Operation mode.
* **Memory Management:** Uses an array of structs to store the state of every question for end-of-game reporting.
* **Visual Feedback:** The console changes color (Green/Red) based on the correctness of the answer.
* **Safe Division:** Implements logic to prevent "Division by Zero" crashes.

---

## 🛠️ Technical Highlights (What I Learned)
This project was a deep dive into structured C++. Key technical implementations include:
* **Enums & Structs:** Used to ensure type safety and organized data grouping.
* **Pass by Reference:** Implemented in functions to efficiently modify the game state without unnecessary memory copying.
* **Ternary Operators:** Used for concise conditional logic and defensive programming.
* **Randomization:** Utilizing `srand()` and `time()` to ensure a unique experience every session.
